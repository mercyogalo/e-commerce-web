# Quick Setup Guide

## Installation Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

- `app/` - All pages (Home, Shop, Product Detail, Login, Signup, Contact, Cart, Checkout, Confirmation)
- `components/` - Reusable UI components
- `contexts/` - React Context (Cart)
- `lib/` - Types and mock data

## Key Features

- **Home Page**: Hero slider, categories, features, promotional banners, top products
- **Shop Page**: Product listing with category filters
- **Product Detail**: Image gallery, color selector, quantity, add to cart
- **Cart**: View items, update quantities, remove items
- **Checkout**: Shipping and payment forms
- **Confirmation**: Order confirmation with receipt

## Notes

- All images use Unsplash placeholders
- Cart data persists in localStorage
- No backend required - all mock data
- Fully responsive and mobile-friendly

## Build for Production

```bash
npm run build
npm start
```
